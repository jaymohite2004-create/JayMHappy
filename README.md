# ReBrew Subscription Interest Streamlit App

This repository contains a Streamlit dashboard for predicting whether a customer is interested in a coffee ground subscription.

## Files
- `app.py` – main Streamlit application.
- `ReBrew_Market_Survey_Synthetic_Data_600_responses.xlsx` – dataset.
- `requirements.txt` – Python dependencies for Streamlit Cloud.

## How to use on Streamlit Cloud
1. Create a new GitHub repository.
2. Upload all files from this zip (no extra folders needed).
3. On Streamlit Cloud, create a new app and point it to `app.py`.
4. The dashboard will load automatically using the included dataset.
